# Grunt Basic Theme

### Installation

It requires [Node.js](https://nodejs.org/) to run.

You need Grunt installed globally:

```sh
$ sudo npm install -g grunt
```

```sh
$ git clone https://github.com/sergiasenjo/Grunt-Basic-Theme.git
$ cd Grunt-Basic-Theme
$ npm install
$ grunt -v
```

### Plugins

  - Uglify
  - Less
  - CSSMin
  - Watch
